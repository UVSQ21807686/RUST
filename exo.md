### Exercice 1 :
     écrire en deux méthode le programme qui calcule la factorielle de 45r et indique à l'écran le résultat.
     


### Correction :
   
    ### méthode 1:
    
    ```markdown
    fn main() {
    let mut n :i32 =5;
    let mut i :i32 =1;
    
    
    loop {
       n=n*i;
       i=i+1;
       if i==5{
           break;
       }
    }
    println!("{:?}",n);
    
}
    ```
